# Temas organizados

Esta carpeta reorganiza la bibliografia en temas de estudio y crea un archivo independiente por cada texto principal.

La idea no es darte una simple ruta de lectura, sino un reemplazo practico en espanol: cada archivo explica de que trata el texto, como esta armado, que tesis defiende, que ejemplos usa y que necesitas retener para estudiar.

Tambien incluye una carpeta de material complementario y una nota sobre los textos que ya estaban disponibles en espanol en la bibliografia original.
