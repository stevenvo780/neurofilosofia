# Visualizaciones y modelos

Esta carpeta contiene diagramas, esquemas formales y bloques en `LaTeX` para consultar rapido relaciones entre conceptos.

La idea es complementar los archivos explicativos con una capa mas visual y mas formal, util si piensas bien en terminos de sistemas, flujo de informacion, niveles, redes o modelos.

## Nota tecnica

- Los diagramas estan escritos sobre todo en `mermaid`.
- Las formulaciones matematicas y logicas estan en `LaTeX`.
- No sustituyen la lectura conceptual; sirven para orientacion, comparacion y compresion estructural.
