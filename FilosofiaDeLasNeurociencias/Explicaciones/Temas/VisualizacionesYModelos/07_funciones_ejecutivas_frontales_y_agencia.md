# Funciones ejecutivas, frontales y agencia

## 1. Estructura del dominio ejecutivo

```mermaid
flowchart TD
    A[Funcion ejecutiva] --> B[Toma de decisiones]
    A --> C[Planificacion]
    A --> D[Resolucion de problemas]
    A --> E[Inicio de accion]
    A --> F[Mantenimiento de accion]
    A --> G[Seleccion de respuesta]
    A --> H[Monitoreo]
    A --> I[Metatasking]
```

## 2. Historia conceptual

```mermaid
flowchart LR
    A[Funciones frontales] --> B[Funciones ejecutivas]
    B --> C[Enfoque ecologico y funcional]
```

## 3. Triparticion clasica de prefrontal

```mermaid
flowchart TD
    A[Prefrontal] --> B[Orbitofrontal]
    A --> C[Cingulada]
    A --> D[Dorsolateral]
    B --> B1[Control social e inhibicion]
    C --> C1[Impulso y conducta dirigida a metas]
    D --> D1[Planificacion y operaciones neuropsicologicas]
```

## 4. Agencia minima como esquema funcional

```latex
\[
\text{Agencia} = f(G, P, S, M)
\]
```

donde:

- \(G\) = generacion de metas,
- \(P\) = planificacion,
- \(S\) = seleccion de respuesta,
- \(M\) = monitoreo.

Si alguno falla gravemente, la agencia se desorganiza.
