# 15a - Nave, Deane, Miller y Clark - Wilding the Predictive Brain

## De que trata

Presenta el marco del procesamiento predictivo en una version amplia, encarnada y afectiva.

## Tesis central

El cerebro puede entenderse como un sistema que genera predicciones y minimiza error, pero ese proceso involucra cuerpo, accion, emocion y entorno, no solo computacion intracraneal.

## Explicacion detallada

El texto comienza con la idea de inferencia perceptiva: el cerebro construye hipotesis sobre las causas del flujo sensorial y usa el error de prediccion para corregirlas.

Despues pasa a inferencia activa. El organismo no solo ajusta su modelo interno; tambien actua para producir el tipo de senal sensorial que espera o que le resulta util.

Un aspecto clave es la precision. El sistema debe estimar cuanto peso darle a ciertas senales y a ciertos errores. Eso conecta el modelo con atencion, valor, relevancia y emocion.

Los autores quieren evitar un neurocentrismo estrecho. Por eso muestran que el procesamiento predictivo encaja bien con mente corporizada, regulacion homeostatica e incluso con extension hacia nichos sociales y tecnologicos.

## Lo mas importante para estudiar

- Procesamiento predictivo = percepcion + accion + emocion.
- El cuerpo y el entorno no son anexos, sino parte del modelo.
- Texto muy importante para unir filosofia de la mente y neurociencia actual.
