# Catedra2026_1PorSesion

Esta carpeta reorganiza el material siguiendo el itinerario real de la catedra:

- una carpeta por fecha y tema;
- dentro de cada sesion:
  - `LecturasOriginalesMd`
  - `LecturasPdf`
  - `Explicaciones`

## Importante

No se movieron ni se borraron los archivos originales de `Bibliografia`.

Lo que se hizo fue:

- crear una estructura nueva orientada al curso;
- enlazar dentro de cada sesion los textos relevantes;
- separar claramente entre lecturas originales y explicaciones.

## Estado actual de los PDFs

En la bibliografia original no aparecieron archivos `.pdf`, por eso todas las carpetas `LecturasPdf` quedaron preparadas pero vacias.
