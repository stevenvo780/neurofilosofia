# Tabla de textos y usos

Esta tabla responde a una pregunta practica: si en clase aparece cierto autor o cierto escrito, para que te sirve y con que otros textos conviene vincularlo.

## Fundamentos

- `Bechtel, Mandik y Mundale`: para definir el campo. Vinculos: Bickle, Bechtel y Huang.
- `Daugman`: para revisar metaforas del cerebro. Vinculos: Chirimuuta, Cobb, Nave.
- `Hinton`: para conexionismo y aprendizaje distribuido. Vinculos: Bickle, Bechtel representaciones.
- `Ray`: para base biologica general. Vinculos: vision, lenguaje, frontales.
- `Bickle`: para Churchland, eliminativismo y reduccion. Vinculos: Hinton, Bechtel, filosofia de la mente.

## Metodo y evidencia

- `Bechtel - epistemologia`: para evidencia, artefacto y tecnicas. Vinculos: Raichle, Mental Mechanisms.
- `Raichle`: para neuroimagen funcional. Vinculos: Bechtel, Dehaene.

## Vision

- `Trivino-Mosquera et al.`: para anatomia y via visual. Vinculos: Zeki, The Mind's Machine.
- `Zeki`: para especializacion funcional. Vinculos: representacion, Raichle.

## Memoria y representacion

- `de Brigard y Robins`: para taxonomias de memoria y huellas. Vinculos: LeDoux, Moser, Bechtel.
- `Quian Quiroga et al.`: para selectividad neuronal y conceptos. Vinculos: representacion, memoria.
- `Bechtel - representaciones`: para concepto filosofico de representacion. Vinculos: Hinton, Zeki, Moser.
- `Moser y Moser`: para mapa espacial y memoria. Vinculos: representacion, hipocampo.

## Lenguaje

- `Baggio`: para marco general de neurolinguistica. Vinculos: Hickok et al., filosofia de ciencias cognitivas.
- `Hickok, Bellugi y Klima`: para lengua de senas y naturaleza del lenguaje. Vinculos: Baggio.

## Emocion, cuerpo y trastorno

- `LeDoux`: para miedo, amigdala y aprendizaje emocional. Vinculos: memoria, Barrett.
- `Chen et al.`: para interocepcion. Vinculos: Barrett, neuropsiquiatria.
- `Barrett`: para emocion, inflamacion y enfermedad. Vinculos: Chen, Ramirez-Bermudez.
- `Ramirez-Bermudez et al.`: para psicopatologia y neuropatologia. Vinculos: Barrett, neuroscience and psychopathology.

## Control, conciencia y accion

- `Suchy`: para funciones ejecutivas y conducta adaptativa. Vinculos: Miller y Cummings, Obhi y Haggard.
- `Miller y Cummings`: para historia y clinica de lobulos frontales. Vinculos: Suchy.
- `Laureys`: para conciencia y estado vegetativo. Vinculos: neuroimagen, persona, etica.
- `Nave et al.`: para cerebro predictivo, cuerpo y entorno. Vinculos: Daugman, Barrett, Chen.
- `Webb`: para modelo mecanicista artificial. Vinculos: explicacion y mecanismos.
- `Obhi y Haggard`: para libre albedrio y control. Vinculos: Suchy, Laureys.
