## Que contiene esta carpeta

La carpeta esta pensada como apoyo de clase, no como resumen para examen. La idea es que puedas:

- ubicar rapido un autor o un texto;
- entrar por tema aunque no recuerdes el titulo del escrito;
- seguir una explicacion en espanol suficientemente fiel;
- encontrar conceptos, cifras, conclusiones y conexiones entre textos;
- moverte rapido cuando el profesor salte de un tema a otro.

## Criterio de rigor de esta carpeta

Cuando el texto original ofrece fechas, cifras o conclusiones concretas, se intentan conservar de forma fiel. Cuando la explicacion es interpretativa o de apoyo para estudiante, esta adaptada para hacerla entendible sin perder la tesis principal.

En general:

- los datos exactos se presentan como datos del texto;
- las conexiones entre textos se presentan como ayuda de estudio;
- las simplificaciones se usan solo para hacer entendible el contenido, no para cambiarlo.
